Usability Test Results

Three Significant Observations

  • Users appreciated the peaceful aesthetic and narrative tone but didn't immediately grasp the future-oriented theme without additional explanation.
  • The clickable elements (NPCs, CTA buttons, dialog close buttons) were not visually distinct, causing confusion about what actions were available.
  • In both mini-game scenes, users struggled to identify the interactive zones, especially due to low contrast or elements blending into the background.

How the Project Can Be Improved

Main Scene:

  • Add a white border or glow around clickable NPCs to indicate interactivity.
  • Redesign CTA buttons and dialog close (“X”) buttons to stand out more clearly with higher contrast or icon styling.
  • Improve visual cues (e.g. hover state, slight movement) to guide user actions.

Women's Game Scene:

  • Brighten the background to visually separate clickable elements from the scene.
  • Add subtle animation (e.g. pulsing or color shift) to indicate clickable objects.
  • Ensure instructions are more clearly labeled so users understand the goal of the interaction.

Elderly Game Scene:

  • Increase contrast between background and buttons to make interaction areas more visible.
  • Highlight draggable elements with borders or shadows.
  • Consider using a tooltip or label to guide first-time users.

Narrative Clarity:

  • Make the theme of “imagining the future” more explicit in the intro scene or dialogue.
  • Add a short written intro or visual cue that helps contextualize the user’s actions as building a future.

Can Users Figure Out How to Navigate/Interact with Your Project?

Yes — after revisions, clearer instructions and highlighted interactive elements were added to help users understand how to navigate the scene and interact with characters, games, and dialogue elements.

Does the Project Impact the User the Way You Want It To?

The project mostly impacts users as intended. It evokes reflection and curiosity, but the future-oriented message needs to be more clearly signposted for full emotional resonance.

What Modifications Need to Be Made to Interactions, Visuals, or Other Elements to Make It Stronger?

  • Strengthen visual hierarchy for buttons and dialog boxes.
  • Improve contrast and add hover/animation cues to draw attention to interactive zones.
  • Reinforce narrative context by adjusting the pacing and placement of thematic text.
  • Animate final scene feedback to create a more engaging and satisfying conclusion.
